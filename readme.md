# Telegram bot to convert pdf/images to Microsoft Word

# How to run 
- Install Python if u don't have.
- Install Pandoc, [installation guide](https://pandoc.org/installing.html)
- Create [Telegram bot](https://core.telegram.org/bots/tutorial) and get Bot Token
- Create [MistralAI](https://docs.mistral.ai/getting-started/quickstart/) account and API-Key
- Run `npm install`
- Run `npm run dev`

# Dependencies 
- Python
- Pandoc
- MistralAI
- Telegram
